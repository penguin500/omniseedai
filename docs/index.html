🔗 API Endpoints

GET /v1/quote
Get a swap quote with OmniSwap fee included. Returns expected output amount, price impact, and route info.
ParameterRequiredDescription
inputYesToken to sell (symbol: SOL, USDC, or mint address)
outputYesToken to buy
amountYesAmount in smallest unit (lamports for SOL = amount * 10^9)
slippageBpsNoSlippage tolerance in bps (default: 50 = 0.5%)
apiKeyNoYour API key for tracking
POST /v1/swap
Build a swap transaction. Returns a base64-encoded Solana transaction ready to sign and send.
Body FieldRequiredDescription
quoteResponseYesQuote object from /v1/quote
userPublicKeyYesSigner's Solana wallet address
wrapAndUnwrapSolNoAuto-wrap/unwrap SOL (default: true)
GET /v1/price
Get current price, volume, and market data for any Solana token.
ParameterRequiredDescription
tokenYesToken symbol or mint address
GET /v1/tokens
List popular tokens or search by name/symbol.
ParameterRequiredDescription
searchNoSearch query (name, symbol, or mint)
POST /v1/register
Register for an API key. Free tier: 1,000 requests/day.
Body FieldRequiredDescription
nameYesYour bot/project name
emailNoContact email
botTypeNoType: trading-bot, agent, dapp, etc.

🐍 Python SDK

Full-featured Python SDK with quote, execute, price lookup, and agent tool definitions.

Install 
pip install omniswap  # Coming soon to PyPI
# For now, download from:
# https://penguin500.github.io/omniseedai/sdk/python/omniswap.py
Get Quote 
from omniswap import OmniSwap

swap = OmniSwap(api_key="your_key")  # API key optional
quote = swap.quote("SOL", "USDC", amount_sol=1.5)

print(f"Input:  {quote['input_human']} SOL")
print(f"Output: {quote['output_human']} USDC")
print(f"Impact: {quote['price_impact']}%")
print(f"Fee:    {quote['platform_fee_bps']} bps")
Execute Swap 
# Auto-sign and send (requires solders: pip install solders)
result = swap.execute(
    "SOL", "USDC",
    amount_sol=1.5,
    private_key="your_base58_private_key"
)
print(f"TX: {result['explorer']}")
Token Price 
sol = swap.price("SOL")
print(f"SOL: ${sol['priceUsd']:.2f} ({sol['priceChange24h']:+.1f}%)")

💫 JavaScript SDK

Works in Node.js and browser. Supports Phantom wallet for browser-based execution.

Browser 
<script src="https://penguin500.github.io/omniseedai/sdk/js/omniswap.js"></script>
const swap = new OmniSwap();
const quote = await swap.quote('SOL', 'USDC', { amountSol: 1.5 });
Node.js 
const { OmniSwap } = require('./omniswap');
const swap = new OmniSwap();

// Quote
const q = await swap.quote('SOL', 'BONK', { amountSol: 0.1 });

// Build unsigned TX for your own signing
const tx = await swap.buildTransaction(q, 'your_wallet_address');

// Execute with Phantom (browser only)
const result = await swap.executeWithWallet('SOL', 'USDC', { amountSol: 1 });
